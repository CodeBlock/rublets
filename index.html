Welcome to Rublets

Rublets is a code evaluation IRC bot which uses selinux's sandbox command to safely allow for evaluation and compilation of code by users in an irc channel.

<CodeBlock> !perl6> print "Hello!";
<rublets> Hello!
<CodeBlock> !io> "Hello world!" println
<rublets> Hello world!
<CodeBlock> !io> "Hello world!" utoh_syntax_error
<rublets>   Exception: Sequence does not respond to 'utoh_syntax_error'
<rublets> <output truncated> https://gist.github.com/2faf508b0415df2b4c76

Setting Rublets up

Unfortunately, Rublets is a bit complex to set up, due to the nature of how it works.

At minimum, Rublets requires: * SELinux (enforcing!) * sandbox (In Fedora, this is found in the policycoreutils-python package) * Interpreters for each language that Rublets supports, or each language that you wish to allow evaluations for. ** See the readme for a list of languages that it can currently evaluate.

Rublets doesn't require anything special beyond this, however. It doesn't need its own server, or even its own user account.

You can daemonize it with nohup, or just do like I do, and run it in tmux or screen.
